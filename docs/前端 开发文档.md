# 前端 开发文档

> 实现者：范棋珈（待定）

## 主要内容

实现一个能够部署的网页程序，能与后端服务器进行交互。

## 网页部分（html 网页）

### 首页

1. 登录、注册界面

### 查票页

1. 查询车票
2. 查询订单记录

### 用户管理页面

1. 查询用户
2. 修改用户信息

### 管理页面

1. 添加、删除列车
2. 查询列车
3. 回滚、删除记录

## 逻辑实现

### 0. 页面的部分刷新

使用异步（AJAX）实现。

### 1. 服务器处理请求

用户端发送 HTTP 请求给服务器，用 Nginx 处理请求后将数据交与后端服务器。

### 2. 后端服务器

目前打算使用 PHP 实现，然后与写的 C++ 后端交互。

### 3. 进程间信息交互

使用 socket.
